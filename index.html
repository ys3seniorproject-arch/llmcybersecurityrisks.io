LLM Cybersecurity Risks

OWASP Top 10 for LLM Applications

This research and educational platform demonstrates five vulnerabilities from the OWASP Top 10 for LLM Applications (2025) through interactive Google Colab notebooks and a Google Sites educational portal. Each module includes a vulnerability demonstration and a corresponding mitigation, covering prompt injection, supply chain risks, improper output handling, system prompt leakage, and misinformation.
